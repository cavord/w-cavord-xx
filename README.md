# w-cavord-xx/Discord

* This is a tool that helps you use Discord webhooks to spam quickly.
> wcavord is a lightweight and fast Discord webhook utility built for educational and testing purposes.
It provides multiple features to send messages, edit webhook info, and manage mass-webhook actions easily.

## 〢 Features
- [x] Send a Message - Quickly send any text to a selected webhook
- [x] Change Webhook PFP - Update the avatar of the webhook
- [x] Change Webhook Name - Modify the webhook display name
- [x] Spam All Webhooks - Send high-speed messages through all saved webhooks
- [x] Delete All Webhooks - Instantly remove every stored webhook
## 〢 Tutorial

1. Download the `.zip` file and run`webhooks.txt` file 
2. Paste your webhook URL inside, then press `Ctrl + S` to save.
3. Open the `.exe` file and start using the tool.

> [!NOTE]
> [2]: Change Webhook PFP — If you want to change the webhook avatar, choose an image you want to use as the new pfp.  
> - The image **must** be in `.png` format.  
> - After that, open the tool, enter command ``2``, and type the exact filename of the image.
